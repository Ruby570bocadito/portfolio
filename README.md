# Rafael Gálvez — Portfolio (v4 · Sophisticated / Threat-Intel)

Cuarta variante: **sofisticada, moderna y corporativa** — el look de una firma top de
ciberseguridad / threat intelligence, con relación al red team pero sin estética "hacker".

- Paleta **obsidiana + carmesí** (rojo = red team, con clase) + platino.
- Tipografía refinada, mucho aire, layout editorial con secciones numeradas.
- **Escena 3D de globo de puntos** con anillos orbitando y nodos satélite (mapa global de amenazas).
- Animaciones suaves: **revelados de texto con máscara**, preloader elegante (marca RG + contador),
  cursor refinado con mix-blend, botones magnéticos, stagger en tarjetas.

Bilingüe ES/EN. **Stack:** Vite · TypeScript · Three.js · GSAP · Vanilla.

---

## 🚀 Desarrollo (Windows / PowerShell)

```powershell
npm install
npm run dev        # http://localhost:3003
npm run build
npm run preview
```

## ✏️ Contenido
Todo en **`src/content.ts`** (idéntico a los otros portfolios). CV en **`public/cv.pdf`**.
Cambia el enlace de LinkedIn por tu slug real.

## 🧑‍🚀 Avatar 3D
Pon tu **`avatar.glb`** (Ready Player Me) en `public/` y sustituye el globo por tu modelo.

## 🌐 GitHub Pages
Incluye `.github/workflows/deploy.yml`. Sube el repo y activa Pages → Source: GitHub Actions.
